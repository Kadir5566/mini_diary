🚀 Key Features (v0)
📁 Hidden Persistence: Automatically creates a concealed .minidiary directory to keep your personal logs organized and private.

⚡ Zero-Memory Indexing: Implements a unique ID generation logic using newline (\n) scanning—achieving data consistency without using any lists or loops.

🛠️ Minimalist CLI: A lightweight Command-Line Interface designed for speed and simplicity.

📝 Smart Append Logic: Efficiently handles file streams to ensure new entries are added instantly without overwriting previous history.

🔒 Constraint-Driven Architecture: Specifically engineered to prove that complex logic (ID tracking/File I/O) is possible under strict programming limitations.
